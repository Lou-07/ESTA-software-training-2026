# 任务列表

## task01 — 提交第二次合作任务项目工程

### 任务目标

将你们在第二次合作任务中完成的项目工程上传到 GitHub，并提交至本培训仓库。

### 任务内容

1. 在自己的 GitHub 账号下创建仓库，将第二次合作任务项目工程推送上去
   - 确保仓库中配置了 `.gitignore`，编译产物（如 `*.o`、`*.exe`、`__pycache__/` 等）不应被提交
2. Fork 本仓库到你自己的 GitHub 账号并 clone 到本地
3. 在 `submissions/<姓名拼音>/task01/` 下创建 `README.md`，内容包含：
   - 你的项目 GitHub 仓库链接
   - 项目简要说明
4. 提交并推送，向本仓库发起 Pull Request，标题格式：`[task01] 姓名`

### 目录结构

```
submissions/<姓名拼音>/task01/
└── README.md
```

### PR 要求

PR 内容请参照 [PR 模板](.github/pull_request_template.md) 填写。
